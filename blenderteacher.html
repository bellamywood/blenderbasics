Blender layout tour

Blender is one application split into areas (think “windows” inside a window).

The 4 key areas
  • 3D Viewport – where you build and edit models.
  • Outliner – list of everything in your scene (organise + select).
  • Timeline – for animation (we’ll mostly ignore today).
  • Properties – where you change object settings (materials, modifiers, etc.).
3D axes (the 3 directions)
  • X = left/right
  • Y = forward/back
  • Z = up/down

Tip: in the Viewport, use the axis gizmo in the top-right to align your view.

✓

Find the 3D Viewport

Largest area. You’ll do most work here.

✓

Use the Outliner to select

Click an object name in the Outliner to select it even if it’s hard to click in 3D.

✓

Spot the Properties panel

This is where you find modifiers, materials, and object settings.

✓

Use the axis gizmo

Top-right of the Viewport. Click the little axis “bubbles” to align views.

✓

Find the 3D cursor

A red/white circle. Be careful: some tools behave differently depending on where your mouse is.

✓

Control tip: move your mouse away from the cursor

Before pressing a shortcut key, keep your mouse near the edge of the Viewport for steadier control.

Controls you must know

These shortcuts are your “power moves”. Learn them now; you’ll use them constantly.

Golden rule

If you start a move/rotate/scale and it goes wrong: Right Click cancels. Ctrl + Z undoes.

✓

Select + confirm + cancel

Left Click confirms an action; Right Click cancels an action.

✓

Move (Grab)

Press G then move your mouse. Tap X, Y, or Z to lock to an axis.

✓

Rotate

Press R then move mouse. Tap X/Y/Z to rotate on one axis.

✓

Scale

Press S then move mouse. Tap X/Y/Z to scale on one axis.

✓

Undo / redo

Ctrl + Z undo, Shift + Ctrl + Z redo.

✓

Orbit / pan / zoom

In the Viewport: scroll middle mouse to zoom. Middle mouse drag to orbit. Shift + middle mouse drag to pan.

✓

Edit Mode selection

In Edit Mode: 1 vertices, 2 edges, 3 faces.

✓

Extrude (in Edit Mode)

Press E to extrude and grow your model.

Build a bowling pin

We’ll start from a cylinder, then extrude + scale to form the pin shape.

Goal

One finished pin that looks smooth, ready to duplicate into a full set of 10.

✓

Clear the scene

If anything is in the scene (like the cube), select it and delete it.

✓

Add the cylinder base

Add → Mesh → Cylinder (Object Mode).

  • Bottom-left “Add Cylinder” panel: set Vertices to 8, Radius to 0.5m, Depth to 0.1m.
✓

Go into Edit Mode

Press Tab (or use the mode dropdown).

✓

Face select mode

Press 3 (Face select). Click empty space to deselect, then select the top face.

✓

Extrude up (start the body)

Press E to extrude upward, then click to confirm.

✓

Scale to shape

Press S to scale in/out. Repeat extrude + scale up the pin.

  • Wider for the body
  • Narrow for the neck
  • Wider again for the head
✓

Round it using edge loops (optional but recommended)

Press 2 (Edge select). Use Shift + Alt + Left Click to select a horizontal edge loop, then S to shape it.

✓

Add Subdivision Surface

Back to Object Mode. Properties → Modifiers → Add Modifier → Subdivision Surface.

✓

Shade Smooth

Right click the pin → Shade Smooth.

✓

Save your work

File → Save into Documents with a sensible name (e.g. bowling_alley.blend).

Want stripes? (Texture painting – optional extension)
  1. Create a 1000×1000 PNG in Paint and save it in Documents.
  2. Material Properties → New, then change the lower-left editor to Shader Editor.
  3. Add an Image Texture node, connect it to Principled BSDF, then open your PNG.
  4. In Edit Mode: press A (select all), UV → Smart UV Project.
  5. Switch to Texture Paint mode and paint stripes.
  6. Save images: Save All Images.

Build a bowling ball

Start from a UV Sphere, then extrude faces inward to create finger holes.

Tip

If you already have your pin in the scene, you can hide it in the Outliner using the eye icon, then reveal it later.

✓

Add a UV Sphere

Object Mode: Add → Mesh → UV Sphere.

✓

Set size + height

Bottom-left “Add” panel: set Location Z to 0.8m, Radius to 0.8m.

✓

Go to Edit Mode

Press Tab.

✓

Face select mode

Press 3. Click empty space to deselect.

✓

Select 3 faces for finger holes

Hold Shift and left click three faces (close together) where fingers would go.

✓

Extrude inward (twice)

Press E and move slightly inward. Repeat so the holes go deeper.

✓

Add Subdivision Surface

Object Mode: add Subdivision Surface and set Levels Viewport from 1 to 2.

✓

Shade Smooth

Right click the ball → Shade Smooth.

✓

Save again

File → Save. If you made textures, remember Save All Images.

Optional extension: colour + texture
  1. Create a 1000×1000 PNG (single colour or a cool pattern).
  2. Material Properties → New → Shader Editor → Image Texture → Open PNG.
  3. If painting: UV → Smart UV Project, then Texture Paint mode.

Build the bowling alley scene

Create the lane, place the pins, then position the ball ready to roll.

Scene-building mindset

Use G to move and X/Y/Z to lock movement. Keep your camera sensible and objects aligned.

✓

Add a lane (Plane)

Object Mode: Add → Mesh → Plane.

✓

Scale the lane

Select the plane → press S to scale. Use S then X or Y to stretch it long like a bowling lane.

✓

Move the pin onto the lane

Select pin → G to move, then Z to lift/drop onto the plane.

✓

Duplicate pins

Select pin → Shift + D to duplicate → move into position.

✓

Make a triangle set (10 pins)

Arrange pins in rows: 4, then 3, then 2, then 1. Use the Outliner to select easily.

✓

Bring in the ball

Reveal the ball (if hidden) and move it to the start of the lane.

✓

Make it yours: add gutters or walls (extension)

Add two long cubes as gutters/walls. Scale and place them along the lane edges.

✓

Materials (simple)

Give the lane a wood-like colour (Material Properties → New → Base Color).

✓

Check your scene

Orbit around. If anything is floating or clipping: select it and press G then Z to fix height.

✓

Final save

File → Save. Screenshot your model for submission if your teacher asks.

Challenge tasks (choose 1–3)
  • Add lane arrows / markings using texture paint.
  • Model a simple “pinsetter” box at the end of the lane.
  • Make a neon sign using emissive material (teacher demo).
  • Animate the ball rolling (Timeline + keyframes) — stretch goal.

Fix common problems

Most Blender problems are “wrong mode”, “wrong selection”, or “lost camera”.

✓

“My shortcut didn’t work”

Click inside the 3D Viewport first. Blender uses the area your mouse is in.

✓

“I can’t extrude”

You must be in Edit Mode and have something selected (face/edge/vertex).

✓

“My object vanished”

Check the Outliner eye icon (object might be hidden). Or press Home after selecting it in the Outliner.

✓

“Everything is orange!”

That means you selected multiple objects/faces. Click empty space to deselect, or use the Outliner.

✓

“My ball holes look sharp”

Use Subdivision Surface and Shade Smooth. Increase Subdivision levels if needed.

✓

“I forgot to save my texture paint”

In Blender: use Save All Images. Then save the .blend file.

Teacher tip: If pupils are stuck, ask: “Are you in Object Mode or Edit Mode?” first.
