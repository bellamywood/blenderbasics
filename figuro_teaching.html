The Main Areas of Figuro

Where things live on the screen

Most of your time will be split across these areas:

  • 3D Viewport: The main stage where you see and edit your model.
  • Tool shortcuts: Use keys like W E R for transform tools and more.
  • Mode shortcuts: Switch what you’re editing (object vs faces/edges/vertices) using A S D F G.
  • On-screen controls: Many tools show a gizmo and/or numeric fields you can tweak.
Behaviour to learn:
Viewport first → choose the correct mode → choose the correct tool → then edit.

Navigation and the 3 Axes

How to move your camera and understand space

Everything in 3D uses three directions (axes):

  • X: left ↔ right
  • Y: forward ↔ backward
  • Z: up ↔ down

Camera controls (mouse):

  • Mousewheel = zoom
  • Drag Right Mouse = orbit around
  • Drag Middle Mouse = pan
  • Shift + Drag Right Mouse = pan
  • Ctrl + Drag Right Mouse = zoom
Rescue tip:
If you “lose” your model, zoom out, then orbit slowly until you find it again.

Keyboard navigation (optional):

  • Arrow keys = navigate camera
  • Arrow keys + Drag Right Mouse = FPS-style navigation

Essential Keyboard Shortcuts

The “must know” set for speed and accuracy

Transform tools:

  • W = Move
  • E = Scale
  • R = Rotate

Modes (what you are editing):

  • A Object mode
  • S Polygon (face) mode
  • D Edge mode
  • F Vertex mode
  • G UV mode (advanced — ignore today)

Editing + workflow:

  • Space = clone (duplicate) selected object(s)
  • Del / Backspace = delete objects or polygons
  • Ctrl + Z = undo • Ctrl + Y = redo
  • Shift + drag (left mouse) = select multiple items
  • Esc or Q = deselect
  • Hold X while moving = snap to grid
Rendering / smoothing (quick preview):
123 change render mode • 4567 change subdivision level • K toggles smooth vs flat shading • L toggles camera light
Teacher notes
  • Most problems today are: wrong mode, or using rotate when they meant move.
  • Make “undo is normal” explicit. Encourage frequent undo instead of panic.

Making the Bowling Pin (Base)

Start from a cylinder primitive
  1. Create a cylinder using the shapes/primitive tools (look for Cylinder).
  2. Low-poly tip: if you see a setting like “Sides/Segments”, set it to 8.
  3. Use W (move) to place it on the “floor”.
  4. Use E (scale) to make it roughly pin-sized (short and chunky to start).
  5. Press S to switch into Polygon (face) mode.
  6. Click the top face so it is selected.

Shaping the Pin

Use the pattern: Extrude → Scale

To build the pin, we repeat one pattern:

Pattern:
T Extrude → then E Scale
  1. With the top face selected: press T to use Extrude.
  2. Pull up a little to extend the body.
  3. Press E to Scale the new top face smaller (start forming the neck).
  4. Repeat: extrude up → scale in → extrude up → scale slightly out for a rounded head.

If your extrudes go sideways: orbit your camera and try again more slowly.

Make it look smooth (quick wins):

  • Press 4–7 to raise the subdivision level (more smoothing).
  • Press K to toggle smooth vs flat shading.

Making the Bowling Ball

Start from a sphere primitive
  1. Switch to Object mode with A.
  2. Create a Sphere (primitive tool).
  3. Use W to place it near the lane and slightly above the floor.
  4. Use E to scale it to a sensible size (smaller than the pin base).
Quick polish:
Try K (smooth shading) and a moderate subdivision level (4 or 5).

Optional: finger holes

  1. Press S for Polygon mode.
  2. Hold Shift and click 3 small faces where holes should go.
  3. Press T (extrude) and push inwards slightly.

Setting Up the Lane

A long plane + a triangle of 10 pins
  1. Switch to Object mode (A).
  2. Create a Plane/Grid primitive for the lane.
  3. Use E (scale) to stretch it into a long lane shape (mostly along Y).
  4. Select your bowling pin.
  5. Press Space to clone (duplicate) it.
  6. Move the new pin into position. Repeat until you have 10 pins in a triangle.
Neat alignment tip:
Hold X while moving to snap to the grid. This helps line up pins neatly.

Final Project Check & Save/Export

Make it presentable and keep your work

Final check:

  • Can you see a lane, pins, and a ball?
  • Is everything sitting roughly on the “floor” (not floating)?
  • Does it look clean from a nice camera angle?
Quick “looks better” buttons:
123 (render mode) • 45 (moderate smoothing) • K (smooth shading)

Make it brighter:

  • Press L to toggle the camera emitting light (on/off).

Save/Export (choose one):

  • Export a model file: look for an Export option and export as OBJ or STL.
  • Take a screenshot: capture your best view if export is blocked.
